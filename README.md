# textodon

텍스트 중심으로 마스토돈을 읽고 쓰기 위한 클라이언트입니다. 여러 계정을 등록해 전환할 수 있으며, 스트리밍으로 타임라인을 실시간으로 반영합니다.

## 시작하기

```bash
bun install
bun run dev
```

## 기능

- 여러 계정 등록 및 전환
- 홈 타임라인 스트리밍 + 무한 스크롤
- 글/댓글 작성 + 공개 범위 선택
- 좋아요/부스트 토글
- 이미지 썸네일 + 팝업 뷰어
- 링크 프리뷰 카드

## OAuth 로그인

1. 계정 섹션에서 마스토돈 인스턴스 주소를 입력합니다.
2. OAuth로 연결을 누르면 해당 인스턴스로 이동합니다.
3. 권한을 승인하면 자동으로 돌아와 계정이 등록됩니다.
4. 승인 시 `read`, `write`, `follow` 권한을 요청합니다.
5. 마지막 사용 시점을 기준으로 7일이 지나면 로컬 저장소의 계정 정보가 초기화됩니다.

## 프로젝트 구조

- `src/domain`: 도메인 모델 정의
- `src/services`: 추상 서비스 인터페이스
- `src/infra`: Mastodon API/스트리밍 구현
- `src/ui`: 화면 구성과 상태 관리

## 배포

GitHub Pages로 배포하도록 설정했습니다. `main` 브랜치에 푸시하면 자동으로 배포됩니다.

- 저장소 Settings → Pages → Build and deployment → Source를 GitHub Actions로 설정합니다.
