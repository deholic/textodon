# textodon

텍스트 중심으로 마스토돈과 미스키를 즐길 수 있는 클라이언트입니다. 여러 계정을 등록해 전환하면서 여러 타임라인 섹션을 동시에 확인하고, 스트리밍으로 실시간 반영합니다.

## 시작하기

```bash
bun install
bun run dev
```

## 기능

- 마스토돈/미스키 계정 등록 및 전환
- 홈 타임라인 스트리밍 + 무한 스크롤
- 여러 타임라인 섹션 + 섹션별 계정 선택
- 섹션 추가/삭제 및 좌우 이동
- 글/답글 작성 + 공개 범위 선택
- 좋아요/부스트/리노트 토글
- 공개 범위 아이콘 표시
- 이미지 인라인 + 확대 뷰어
- 링크 미리보기 카드
- 피드 프로필 이미지 표시 옵션(설정에서 토글)
- 섹션 크기 설정(작게/중간/크게)
- 크리스마스 테마

## OAuth 로그인

1. 사이드바의 서버 추가 버튼을 눌러 인스턴스 주소를 입력합니다.
2. OAuth로 연결을 누르면 해당 인스턴스로 이동합니다.
3. 권한을 승인하면 자동으로 돌아와 계정이 등록됩니다.
4. 승인 권한: `read`, `write`, `follow`
5. 마스토돈 정책상 7일 이상 미사용 시 로컬 저장소의 계정 정보가 초기화될 수 있습니다.

## 프로젝트 구조

- `src/domain`: 도메인 모델 정의
- `src/services`: 추상 서비스 인터페이스
- `src/infra`: Mastodon/Misskey API, 스트리밍 구현
- `src/ui`: 화면 구성 및 상태 관리
- `src/ui/styles`: base/layout/components/theme 분리

## 배포

GitHub Pages로 배포되도록 설정되어 있습니다. `main` 브랜치에 푸시하면 자동으로 배포됩니다.

- 저장소 Settings -> Pages -> Build and deployment -> Source를 GitHub Actions로 설정
